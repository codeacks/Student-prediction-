# 🎓 Student Performance Prediction using Machine Learning

## 📌 Problem Statement
The objective of this project is to predict whether a student will pass or fail based on academic and personal attributes such as study hours, attendance, previous grades, and extracurricular activities.

---

## 📊 Dataset
The dataset contains the following features:
- Study Hours per Week
- Attendance Rate
- Previous Grades
- Participation in Extracurricular Activities
- Parent Education Level
- Target Variable: Passed (Yes/No)

---

## ⚙️ Approach

1. Data Cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Encoding categorical variables
4. Model training using:
   - Logistic Regression
   - Random Forest Classifier
5. Model evaluation using accuracy, ROC-AUC, and confusion matrix
6. Hyperparameter tuning using GridSearchCV
7. Cross-validation for robustness

---

## 🤖 Models Used

| Model                | Purpose                  |
|---------------------|------------------------|
| Logistic Regression | Baseline model         |
| Random Forest       | Non-linear modeling    |

---

## 📈 Results

- Random Forest outperformed Logistic Regression
- Achieved high accuracy and better generalization
- Important features include:
  - Previous Grades
  - Study Hours
  - Attendance Rate

---

## 🔍 Key Learnings

- Importance of proper preprocessing
- Handling categorical data using encoding
- Model comparison and evaluation
- Hyperparameter tuning improves performance
